Trabalho de Conclusão de Curso

Nesse projeto colocamos em prática os conteúdos abordados na Escola da Nuvem sobre os fundamentos da plataforma Amazon Web Services

Tendo como objetivo o desenvolvimento de um website (html, css, imagens) contendo os currículos do grupo e a implantação de um ou mais serviços na nuvem AWS, com rede de entrega de conteúdo, com baixa latência, com segurança, e armazenamento durável, para atender a demanda de 11 milhões de requisições¹ por mês.

Tecnologias

Route 53: é um serviço de DNS (Domain Name System) altamente disponível e dimensionável. Objetivo: roteamento de DNS e verificação de integridade.

CloudFront: é uma rede de entrega de conteúdo (CDN) que acelera à entrega de conteúdo estático e dinâmico da Web para usuários finais. Objetivo: acelerar à entrega de conteúdo para os usuários finais

Amazon Simple Storage Service (Amazon S3): é um serviço de armazenamento de objetos que oferece escalabilidade, disponibilidade de dados, segurança e performance. Você pode utilizar o Amazon S3 para armazenar e recuperar qualquer volume de dados, a qualquer momento, de qualquer lugar na Web. Objetivo: armazenar os currículos e backup do Banco de Dados

AWS Certificate Manager (ACM): é um serviço para provisionar, gerenciar e implantar certificados SSL/TLS públicos e privados para uso com serviços da AWS e seus recursos internos conectados. O ACM elimina processos manuais demorados, como compra, upload e renovação de certificados SSL/TLS. Objetivo: implantar um certificado para a nossa aplicação
